# Notebook App

This is a training project on Ruby on Rails from an Udemy Course. 

Things you may want to cover:

* Ruby version '2.1.2'

* Rails version 4.2.5'

* System dependencies
  bundler must be version >= 1.8.4

* Configuration
  ```sh
  $ bundle install
  ```
* Database creation
  ```sh
  $ rake db:drop db:create
  ```
* Database initialization and seeding
  ```sh
  $ rake  db:migrate db:seed utils:seed
  ```
